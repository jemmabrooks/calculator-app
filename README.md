# Calculator App

A minimal iOS-style calculator built with React and Vite.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage (`%`) and sign toggle (`±`)
- Decimal input with duplicate-dot guard
- Division-by-zero protection (displays `Error`)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

Output goes to `dist/`.
